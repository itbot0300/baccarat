# 百家乐游戏 - 产品文档

## 1. 项目概述

### 产品定位
一款基于浏览器的经典百家乐纸牌游戏，支持真钱金币下注功能，提供流畅的游戏体验和历史记录功能。

### 目标用户
- 喜欢经典纸牌游戏的玩家
- 希望在休闲时间进行娱乐的用户
- 喜欢手机/电脑跨平台游戏的用户

---

## 2. 功能需求

### 2.1 核心游戏功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 发牌系统 | 随机生成庄/闲两家牌面，按百家乐规则计算点数 | P0 |
| 下注区域 | 支持庄家(Banker)、闲家(Player)、和局(Tie)三种下注 | P0 |
| 金币系统 | 玩家初始金币 1000，每次下注扣除，胜利时按赔率返还 | P0 |
| 比牌逻辑 | 庄闲两家比点数大小，点数相同为和局 | P0 |
| 赔率规则 | 庄家赢 1:0.95（抽水5%）<br>闲家赢 1:1<br>和局 1:8 | P0 |

### 2.2 游戏历史

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 记录展示 | 显示最近 20 局游戏结果（庄/闲/和） | P1 |
| 数据统计 | 统计庄/闲/和的胜率走势 | P1 |
| 本地存储 | 使用 localStorage 保存历史记录 | P1 |

### 2.3 用户界面

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 金币余额 | 实时显示当前金币数量 | P0 |
| 下注金额 | 可选择下注金额（10/50/100/500） | P0 |
| 结算动画 | 发牌、比牌、结算时的视觉反馈 | P1 |
| 音效提示 | 下注、发牌、结算时的音效（可选） | P2 |

## 3. 技术规格

### 3.1 响应式设计

| 屏幕尺寸 | 布局 |
|----------|------|
| 桌面 (>768px) | 横向布局，牌面居中，下注区在下方 |
| 移动端 (<=768px) | 纵向布局，牌面自适应宽度 |

### 3.2 数据存储

```
localStorage:
  - baccarat_balance: 当前金币余额
  - baccarat_history: 游戏历史记录 (JSON数组)
  - baccarat_stats: 统计数据
```

## 5. 百家乐规则说明

### 点数计算
- A = 1 点
- 2-9 = 面值点数
- 10/J/Q/K = 0 点

### 比牌规则
1. 闲家先补第三张牌（如需要）
2. 庄家根据闲家第三张牌决定是否补牌
3. 庄家补牌规则：
   - 0-2点：必须补牌
   - 3点：闲家补8则不补，其他补牌
   - 4点：闲家补0/1/8/9则不补，其他补牌
   - 5点：闲家补0/1/2/3/8/9则不补，其他补牌
   - 6点：闲家补6/7则不补，其他补牌
   - 7点：停牌
   - 8-9点：天然停牌
