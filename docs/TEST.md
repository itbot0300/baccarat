### 3. 玩家最多3张牌 ✅

**测试方法：** 验证后端补牌逻辑

**后端补牌逻辑 (app.py):**
```python
# 闲家补牌规则
if player_score <= 5:
    player_third_card = self.draw_card()
    player_cards.append(player_third_card)
    player_score = self.calculate_score(player_cards)
```

**分析：**
- 初始发2张牌
- 只有在点数 <= 5 时才补第三张牌
- 最多只能补1张牌
- 玩家最多3张牌 ✅

**结论：** ✅ 通过 - 代码逻辑正确实现
