# 百家乐游戏 - 产品路线图

## 版本规划

### v10 - MVP（.0.最小可行产品）
**目标：完成核心游戏功能**

- [ ] 基础游戏界面搭建
- [ ] 百家乐发牌与比牌逻辑
- [ ] 庄/闲/和三种下注
- [ ] 金币系统（余额/下注/结算）
- [ ] 赔率计算（庄1.95/闲1.0/和8.0）
- [ ] 基础响应式布局
- [ ] 移动端和桌面端测试

**预计开发时间：2-3 天**

---

### v1.1.0 - 历史记录功能
**目标：增加游戏历史与统计**

- [ ] localStorage 存储游戏历史
- [ ] 显示最近20局结果
- [ ] 庄/闲/和胜率统计
- [ ] 简洁的历史记录UI

**预计开发时间：1 天**

---

### v1.2.0 - 体验优化
**目标：提升用户体验**

- [ ] 发牌动画效果
- [ ] 结算动画反馈
- [ ] 音效支持（可开关）
- [ ] 更好的视觉效果
- [ ] 断线重连/数据恢复

**预计开发时间：2 天**

---

### v2.0.0 - 高级功能（可选）
**目标：增加游戏趣味性**

- [ ] 多人对战场合（观战系统）
- [ ] 番摊/龙虎等扩展游戏
- [ ] 排行榜系统
- [ ] 成就系统

**预计开发时间：待定**
